# 🌦️ SkyView Weather App

SkyView is a modern and responsive weather application built using **HTML, CSS, and JavaScript**. It provides real-time weather updates for any city along with current location detection and a clean, user-friendly interface.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 📍 Get weather using current location  
- 🌡️ Real-time temperature display  
- ☁️ Weather condition with icons  
- 💧 Humidity and 🌬️ wind speed  
- 📅 Short-term forecast (next 2 days)  
- 🎨 Modern UI with animations (Glassmorphism design)  
- 📱 Fully responsive (mobile-friendly)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Weather API (OpenWeather)
